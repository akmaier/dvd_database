# Requirements Analysis — DVD Shelf Database

This folder contains the requirements analysis and design documentation for the DVD Shelf Database application.

## Documents

| Document | Description |
|----------|-------------|
| [Requirements Specification](requirements-specification.md) | Functional and non-functional requirements, data model, constraints |
| [Use Cases](use-cases.md) | Use case diagram (Mermaid) and detailed use case descriptions |
| [Sequence Diagrams](sequence-diagrams.md) | Interaction flows for page load, filtering, detail view, and error handling |
| [Structural Diagrams](structural-diagrams.md) | Component diagram, data model, DOM structure, and file layout |

## Diagram Format

All diagrams use [Mermaid](https://mermaid.js.org/) syntax embedded in Markdown. GitHub renders these natively — no plugins or image files needed.
